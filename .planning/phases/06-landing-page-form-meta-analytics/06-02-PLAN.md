## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| browser-API → DOM | `Intl.DateTimeFormat().resolvedOptions().timeZone` is browser-provided (CLDR data); not user-controlled. Treated as trusted scalar for `textContent` (not `innerHTML`). |
| URL → DOM | `?error=` query param IS user-controlled (anyone can craft a URL). Mitigation via lookup-table COPY map + `textContent` (not `innerHTML`). |
| browser → plausible.io | Outbound event POST (sendBeacon under the hood). CSP `connect-src https://plausible.io` allow-lists it. |
