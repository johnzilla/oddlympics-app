## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Author -> repo | Hand-translated copy from `references/privacy.md` into `src/pages/privacy.astro`. Untrusted in the sense that copy errors can leak prohibited terms or wrong disclosures. |
| Build pipeline -> served HTML | Astro prerender emits the final static HTML that paid-ad reviewers and end users will GET. Whatever lands in `dist/client/privacy/index.html` is what ships. |
| User browser -> page | One-way: browser requests `/privacy`, receives static HTML. No form, no JS, no cookies, no inputs accepted from the user. |
